# React + TypeScript + Vite + Material UI Template

A clean and scalable frontend starter template built with modern react.

## Features

- ⚡ Vite + React + TypeScript setup
- 🎨 Material UI with customizable theme
- 🌙 Light / Dark mode toggle
- 🔔 Global alert system (Snackbar)
- ⏳ Global loading state management
- 🧱 Clean and scalable folder structure
- 🧭 React Router with layout support
- 🧩 Reusable components (Navbar, Footer, etc.)

## Project Structure

Organized for scalability and maintainability:

- `components/` – Reusable UI components
- `pages/` – Route-level components
- `layouts/` – App layout (Navbar, Footer)
- `theme/` – Theme configuration and context
- `utils/` – Global utilities (alerts, loading)

## 🛠️ Getting Started

```bash
npm install
npm run dev
```

Feel free to extend it based on your project needs.
